# gatekeepeer

Бот, который сам принимает заявки на вступление в ваш приватный Telegram-канал
или группу. Включили приём по заявкам, чтобы отсеять ботов и спам — но теперь
кто-то должен вручную жать «принять» под каждой заявкой? Добавьте gatekeepeer, и
он будет делать это за вас, мгновенно и круглосуточно.
